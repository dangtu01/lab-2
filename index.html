HTML

HTML is the fundamental building block of webpages

What is HTML?

  1. Markup Language: HTML is a markup language, not a programming language.
  2. Tags: HTML uses tags to define elements on a webpage.
  3. Document Structure:HTML documents have a hierarchical structure.
  4. Semantic Markup:HTML5 introduced a focus on semantic markup, where elements are used to convey the meaning or purpose of the content.
